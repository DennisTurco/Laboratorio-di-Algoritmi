Detailed Description

consegna1 per il corso "Laboratorio di Algoritmi".

Author
DENNIS TURCO
Version
1.0.0
consegna1_DennisTurco.c

compilazione: g++ -xc++ consegna1_DennisTurco.c
run: ./a.out

Il programma carica il file data.txt contenente 107 righe con i dati di contagi covid per provincia. In output.h viene mostrato il numero di accessi in read alla memoria per eseguire il sorting di ciascuna riga.

Note
Obiettivo: Creare un algoritmo di sorting che minimizzi la somma del numero di accessi per ogni sorting di ciascuna riga del file. Le informazioni riguardanti il numero di accessi e' possibile consultarle nel file output.h

Algoritmo di sorting "QuickSort"

utilizzo l'algoritmo di QuickSort per ordinare solo gli elementi maggiori (strettamente) del pivot, posizionati in maniera "disordinata" quindi non sono ordinati (o parzialmente ordinati) ne in modo crescente ne in modo decrescente, quindi mi aspetto di ottenere un costo computazionale di circa Θ(n(log(n))).
