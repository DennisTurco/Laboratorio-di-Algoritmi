Consegne

Consegna1

La consegna prevede di sviluppare, partendo dal file consegna1-loader.c, un algoritmo di sorting dedicato per la serie di dati data.txt (dati di contagi giornalieri per provincia italiana durante l'epidemia covid-19). In particolare si deve sviluppare un unico algoritmo di ordinamento che sar√† eseguito sui dati di ciascuna provincia. L'obiettivo e' quello di minimizzare le letture in memoria durante l'ordinamento delle serie di dati. Consegnare il file c e un file di testo in cui si riporta l'output del programma (numero di letture in memoria per ciascuna serie e i totali).

Consegna2

Completare il file lezione11-1-tree-dot-eulero.c:

  1. euler tour: stampa e ricostruzione albero da lista nodi partendo dallo scheletro visto a lezione, completare la funzione ricorsiva per la creazione dell'albero, basandosi sul file di input contenente una visita di eulero. L'albero ottenuto deve produrre una visita di eulero identica al file di input;
  2. flip albero: dato un albero in input, produrre un nuovo albero in cui tutti i figli destri e sinistri risultano scambiati;
  3. height - depth: calcolare la profondita' o l'altezza dell'albero.

Consegna3

Partendo dal codice del max Heap, costruisci un algoritmo per fare heapsort:

  1. trasforma max heap in min heap (con le operazioni heap_insert e remove_min, non serve decrease_key);
  2. crea l'algoritmo per il sorting di un array random in input;
  3. misura le letture in memoria e confronta con una implementazione di quicksort (ricicla il codice di quicksort con il lancio su 10 array di dimensione da 100 a 1000, ripeti 1000 volte con array casuali);
  4. produci un grafico del confronto letture quicksort - heapsort. Consegna il codice c e il foglio xls con i grafici di confronto.

Consegna4

  1. Implementare la rappresentazione del grafo tramite matrice di adiacenza;
  2. Modificare il codice lezione15-grafi e sostituire le liste di adiacenza con la matrice;
  3. Adattare la creazione del grafo e la visita Dfs per lavorare con le matrici;
  4. Mantenere il codice originale, non sono accettate implementazioni completamente riscritte.

Consegna5

Partendo dal codice lezione17-grafi-dijkstra.c

  1. implementare l'algoritmo di Dijkstra utilizzando il min-heap precedentemente sviluppato;
  2. lanciare l'algoritmo su un grafo che rappresenta un quadrato NxN sul piano cartesiano (ogni nodo i corrisponde alla posizione (x,y) con i=x+N*y). Gli archi del grafo connettono nodi a distanza euclidea = 1.
